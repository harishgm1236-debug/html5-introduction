What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages using different tags.

Basic HTML Syntax

HTML documents start with <!DOCTYPE html> and contain elements wrapped inside tags.

Common HTML Tags

  • <html> – Root element
  • <head> – Metadata
  • <title> – Page title
  • <body> – Page content
  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <ul> <li> – Lists

HTML5 Semantic Tags

HTML5 introduced semantic tags like:

  1. <header>
  2. <section>
  3. <footer>
